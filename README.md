# 💙 Website GMKI Surabaya

Website resmi Gerakan Mahasiswa Kristen Indonesia (GMKI) Cabang Surabaya yang dikembangkan untuk menyediakan informasi organisasi, kegiatan, program kerja, dokumentasi, serta media komunikasi antara pengurus, anggota, calon anggota, dan masyarakat umum.

🌐 Website: https://web-gmki-surabaya.vercel.app/

💻 Repository: https://github.com/marioklau/Web-GMKI-Surabaya

---

## 📖 Tentang GMKI

Gerakan Mahasiswa Kristen Indonesia (GMKI) merupakan organisasi kader yang berlandaskan nilai-nilai kekristenan, keilmuan, dan pengabdian kepada masyarakat. GMKI hadir sebagai wadah pembinaan mahasiswa Kristen agar mampu menjadi pemimpin yang berintegritas, kritis, dan berkontribusi bagi gereja, perguruan tinggi, serta bangsa Indonesia. GMKI Surabaya aktif melaksanakan berbagai kegiatan kaderisasi, diskusi kebangsaan, pengabdian masyarakat, dan pelayanan kemahasiswaan. :contentReference[oaicite:0]{index=0}

## 🎯 Tujuan Website

Website ini dikembangkan untuk:

- Menjadi pusat informasi resmi GMKI Surabaya.
- Mempermudah penyebaran informasi kegiatan dan program kerja.
- Menyediakan media publikasi dokumentasi organisasi.
- Mempermudah calon anggota mengenal GMKI Surabaya.
- Meningkatkan keterbukaan informasi organisasi.
- Mendukung transformasi digital organisasi kemahasiswaan.

---

## ✨ Fitur Utama

### 🏠 Beranda
Menampilkan informasi singkat mengenai GMKI Surabaya, visi organisasi, dan informasi terbaru.

### 📢 Informasi Kegiatan
Menyediakan informasi kegiatan yang telah maupun akan dilaksanakan oleh GMKI Surabaya.

### 🖼️ Dokumentasi
Menampilkan galeri foto kegiatan organisasi.

### 👥 Profil Organisasi
Berisi sejarah, visi, misi, dan struktur organisasi GMKI Surabaya.

### 📅 Program Kerja
Menampilkan program kerja dan agenda organisasi.

### 📞 Kontak
Menyediakan informasi kontak yang dapat dihubungi oleh anggota maupun masyarakat umum.

### 📱 Responsive Design
Tampilan website dapat diakses dengan baik melalui desktop, tablet, maupun perangkat mobile.
